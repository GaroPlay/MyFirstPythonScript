# 🧮 Python Tkinter Calculator

Простой и элегантный калькулятор, написанный на Python с использованием библиотеки `tkinter`. Проект демонстрирует создание графического интерфейса (GUI), обработку логики вычислений и упаковку скрипта в исполняемый файл `.exe`.

![Version](https://img.shields.io/badge/version-1.0-blue.svg)
![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

## 📚 Используемые технологии

- **Python** — язык программирования.
- **Tkinter** — стандартная библиотека для создания GUI.
- **PyInstaller** — инструмент для упаковки скриптов в `.exe`.
